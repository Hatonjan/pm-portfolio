# Portfolio Website - Jonathan Scott

A modern, responsive portfolio website showcasing professional experience in construction management and project management, with a growing focus on computer science and technology.

## 🌐 Live Demo

[View Live Site](https://pm.mjscott.dev)

## 📋 About

This is a personal portfolio website built from scratch without external libraries or frameworks. It features a clean, professional design highlighting 17 years of construction industry experience, educational achievements, certifications, and project work.

## ✨ Features

- **Responsive Design**: Fully responsive layout that works seamlessly across desktop, tablet, and mobile devices
- **Hamburger Menu**: Custom-built mobile navigation menu with smooth animations
- **Infinite Image Slider**: Auto-scrolling project gallery with hover-to-pause functionality
- **Semantic HTML5**: Proper use of semantic tags for accessibility and SEO
- **Custom Animations**: Smooth transitions and hover effects throughout
- **Contact Form**: Integrated Netlify form for direct communication
- **No Dependencies**: Built entirely with vanilla HTML, CSS, and JavaScript

## 💻 Local Development

1. Clone the repository:
```bash
git clone https://github.com/Hatonjan/pm-portfolio.git
```

2. Navigate to the project directory:
```bash
cd pm-portfolio
```

3. Open `index.html` in your browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

4. View in browser at `http://localhost:8000`
